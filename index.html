Objetivos

Este curso foi pensado para quem está dando os primeiros passos no mundo do desenvolvimento web. Nosso principal objetivo é apresentar os conceitos fundamentais de forma simples e prática, ajudando você a entender como funcionam as páginas da internet. Aqui, você vai desenvolver o raciocínio lógico, aprender a escrever suas primeiras linhas de HTML e CSS, e conhecer ferramentas que fazem parte do dia a dia de quem cria sites. Tudo isso com atividades que tornam o aprendizado mais leve e envolvente.

Entender o HTML

Aprender a estruturar páginas com as principais tags e boas práticas de marcação.

Nesta etapa, o aluno aprenderá o que é HTML, como utilizar as principais tags, entender a estrutura semântica de uma página e construir páginas simples com cabeçalhos, parágrafos, listas, links, imagens e tabelas. Também serão apresentadas boas práticas para manter o código organizado e acessível.

Estilizar com CSS

Aplicar estilos visuais às páginas usando seletores, propriedades e layouts responsivos.

O módulo de CSS ensinará como transformar uma página simples em um layout atrativo. Os alunos vão aprender sobre seletores, cores, fontes, espaçamentos, posicionamento, responsividade e a utilização de classes. Também será introduzido o modelo de caixa (box model) e técnicas de media queries para diferentes tamanhos de tela.

Visualizar e compartilhar

Interpretar projetos no Figma e publicar o código no GitHub para colaboração.

Nesta fase final, os alunos irão explorar como visualizar protótipos no Figma, extrair referências de design e aplicar no HTML/CSS. Além disso, aprenderão a criar repositórios no GitHub, versionar seus projetos, publicar com GitHub Pages e colaborar com outros desenvolvedores. A ideia é fechar o ciclo com noções básicas de design e compartilhamento de código.

Aulas

Nesta trilha, você vai explorar os principais fundamentos do desenvolvimento web de forma prática e acessível. Cada aula foi pensada para apresentar os conceitos passo a passo, com exemplos visuais, linguagem simples e atividades que te ajudam a fixar o conteúdo. Começamos pela introdução à área, seguimos com HTML e CSS, e depois passamos por ferramentas essenciais como Figma e GitHub. Tudo isso para que você possa criar seus próprios projetos com autonomia e confiança.
